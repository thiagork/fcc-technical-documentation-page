C

Bellow an example of Hello World using the C language.

C is the language that made “Hello, World” examples popular.

                    
                    #include <stdio.h>
                    
                    int main(void)
                    {
                        printf("hello, world\n");
                    }

Interesting facts about C:

  • It was developed and released around the year 1972.
  • Most popular Linux Kernel and most UNIX utilities, what you would call the operating system are written in C.
  • C is not an object oriented language so you don’t get distracted by things like inheritance and polymorphism.
  • The C language has formed the basis for many languages including C++, Java, JavaScript, Go, Rust, Limbo, LPC, C#, PHP, Python, Perl, Verilog, and C-shell.

Haskell

Bellow an example of Hello World using Haskell.

Haskell is a strongly-typed, purely functional programming language.

                    
                    module Main where
                    
                    main :: IO ()
                    main = putStrLn "Hello, World!"

Interesting facts about Haskell:

  • It is named after logician Haskell Curry.
  • Unlike the other languages listed in this website, Haskell is a purely functional.
  • It is lazy evaluated, which means it never calculates anything until it needs to.

Java

Bellow an example of Hello World using Java.

Something that made Java special is that it was designed so you could write code once and then allow it to run on any operation system

                    
                    class HelloWorldApp {
                        public static void main(String[] args) {
                            System.out.println("Hello World!");
                        }
                    }

Interesting facts about Java:

  • Sun Microsystems created Java. Java was originally called Oak. Oracle now oversees Java after its acquisition of Sun in 2010.
  • Java is an enterprise favorite. Some say it runs up to 95% of enterprise desktops.
  • Java was originally designed for interactive TV, but the digital cable industry wasn’t ready for the technology.

Python

Bellow an example of Hello World using Python.

Python has a compact syntax needing far fewer lines of code than languages like Java or C. It's is the fastest growing language currently.

                    
                    print("Hello World")

Interesting facts about Python:

  • Python uses indentation instead of braces to structure its programs and scripts into blocks.
  • According to the creator Guido Van Rossum, the name of this language was derived from the British comedy series “Monty Python’s Flying Circus”.
  • Unlike other programming languages, Python does not support pointers. Instead, objects are passed by reference.
